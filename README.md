
# Uuidy
Async and sync uuid generator. Generation of [RFC4122](http://www.ietf.org/rfc/rfc4122.txt) UUIDs.
ES5 browser compatiable non sync version uses promises.

### Install
- `npm i uuidy --save`

## Api

### Uuidy
- Version 4
    - `four: AsyncFunction`
    - `fourSync: Function`
